# Second Opinion Protocol

## When to Request a Second Opinion
- Diagnostic uncertainty after initial evaluation
- Complex or rare conditions (e.g., atypical uveitis, unusual retinal findings)
- Surgical planning for high-risk cases
- Treatment failure or unexpected progression
- Medicolegal considerations
- Patient request

## Case Submission Checklist
1. **Patient demographics** (anonymized): Age, gender, relevant systemic history
2. **Chief complaint** and duration
3. **Best corrected visual acuity** (BCVA) for each eye
4. **IOP** measurements
5. **Anterior segment findings** (slit lamp)
6. **Posterior segment findings** (fundoscopy)
7. **Imaging**: Attach relevant images (OCT, visual fields, topography, angiography)
8. **Working diagnosis / hypothesis**
9. **Specific questions** for the consultant
10. **Urgency level**: Normal, Urgent (24h), Emergency (immediate)

## Specialist Routing
- **Retina**: DMRI, DR, RVO, retinal detachment, macular pathology, intravitreal injections
- **Glaucoma**: IOP elevation, optic nerve damage, visual field loss, angle closure
- **Cornea**: Keratoconus, corneal infections, transplant evaluation, ectasia
- **Refractive**: LASIK/PRK candidacy, ICL evaluation, refractive surprises
- **Oculoplastics**: Lid pathology, lacrimal disorders, orbital disease
- **Strabismus**: Eye misalignment, diplopia, pediatric alignment
- **Neuro-ophthalmology**: Optic neuritis, papilledema, cranial nerve palsies, visual field defects with neurological pattern
- **Uveitis**: Inflammatory conditions, autoimmune eye disease

## Response Protocol
1. Review all submitted materials thoroughly
2. Document your analysis systematically
3. Provide differential diagnosis (ranked by likelihood)
4. Recommend additional tests if needed
5. Suggest treatment plan with rationale
6. Note confidence level (high/moderate/low)
7. Cite relevant guidelines or literature when applicable
8. If AI-assisted analysis was used, mark opinion as such

## Discussion Guidelines
- Be professional and constructive
- Reference specific findings when discussing
- If disagreeing with referring doctor, explain reasoning clearly
- Recommend follow-up timeline
- Document if case warrants in-person evaluation
